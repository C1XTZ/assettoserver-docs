# RaceChallengePlugin
Plugin to challenge other players to races by flashing your headlights three times

:::note

Do not use RaceChallengePlugin and PatreonRaceChallengePlugin at the same time!

:::

## Configuration
Enable the plugin in `extra_cfg.yml`
```yaml
EnablePlugins:
  - RaceChallengePlugin
```
